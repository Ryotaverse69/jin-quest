# JID×QUEST

スーパーファミコン風RPGゲーム

## 概要

「JID×QUEST」はJID Corporationを舞台にしたレトロRPGです。プレイヤーは新人営業として入社し、井坂会長と梅田社長の下で成長していきます。

## 特徴

- **レトロな解像度**: 256x224 (スーパーファミコン標準)
- **ドット絵スタイル**: 16x16ピクセルのタイルベース
- **ターン制バトル**: ドラクエ風のコマンドバトルシステム
- **役職昇進システム**: レベルアップで役職が上がる
- **JID本社探索**: 4階建ての本社ビルを探索

## キャラクター

### メインNPC
- **井坂会長**: JIDの創設者。プレイヤーにクエストを与える
- **梅田社長**: JIDの女帝。プレイヤーを励まし導く
- **ポメ吉**: 社長の愛犬。セーブポイントとして機能

### 仲間
- 営業事務（サポート）
- 総務（タンク）
- 人事（ヒーラー）
- 経理（デバッファー）
- 審査（アタッカー）
- 求償（特殊アタッカー）

## インストール

### 必要要件
- Python 3.8以上
- Pygame 2.5以上

### セットアップ

```bash
# 依存パッケージをインストール
pip install -r requirements.txt

# ゲームを起動
python main.py
```

## 操作方法

### タイトル画面
- **Enter**: ゲーム開始
- **ESC**: 終了

### フィールド
- **矢印キー / WASD**: プレイヤー移動
- **Space / Enter**: 目の前のNPCに話しかける
- **I**: 情報表示の切り替え
- **ESC**: タイトル画面に戻る

### バトル
- **矢印キー / WS**: コマンド選択
- **Enter / Space**: コマンド決定/メッセージ送り
- バトル終了後: Enterキーでフィールドに戻る

    ├── dialogues/    # 会話データ
